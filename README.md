## Creating Recipe



You can create recipe by pressing the tab Recipes and than on the button Create then you will see this winndow :


![image](https://github.com/Food-Discovery/Food_Discovery/assets/97468479/6e2c2e80-9267-48bd-aed1-046ca5677f26)


##### There you can:
#### Name the  recipe 
#### Write the ingridients  
#### Write the instrutions for prepairing 
#### The time needed for the recipe to be made  
#### Servings that will be produced  
#### Calories for each serving  
#### Authot of the recipe  
#### Type of the recipe (brekfast,lunch,dinner)
